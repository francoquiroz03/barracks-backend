## Requirements

* Node 16
* Git
* MongoDB
* Docker

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/francoquiroz03/barracks-backend.git
cd barracks-backend
```

```bash
docker-compose build
docker-compose up -d
```

## Steps for access

* MongoDB - Robo 3T: FROM URI = mongodb://localhost:27017/barracks
* NodeJS port 8000
